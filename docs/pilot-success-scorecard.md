## Go / No-Go

**Extend pilot** if: >80% triage in TrustOps, weekly QA rhythm, client portal adoption, no tenant-isolation issues.

**Pause** if: integration blocks case creation, client data boundary failure, analysts bypass TrustOps routinely.
